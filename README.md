<h1> 7DaysOfCode </h1>
<p>Desafio 4 de 7: Juego Adivina Número Secreto</p>

Se agregan 2 opciones de juego, paro lo cual se debe cambiar el nombre del archivo .js en archivo .html.

En la opción app-fijo.js: El número secreto es definido manualmente.
En la opción app-aleatorio.js: El número secreto es definido aleotariamente.

Para ambos casos el rango de números secretos es entre 0 y 10.
En cada juego tienes la opccion de adivinar en 3 oportunidades.

Suerte y disfruta del juego.
